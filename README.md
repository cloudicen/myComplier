# myComplier
可配置的词法语法分析器 -- 用于函数绘图语言编译器

## 架构综述

```
可配置的词法语法分析器
|
├── 表驱动语法分析器
|    └── 预测分析表 （向语法单元添加预测规则后生成的表）
│        └──语法单元 (parseElement class)
|
└── NFA|DFA混合驱动的词法分析器
    |── 词法定义列表(一系列定义好的词法单元组成的列表）
    |    └── 词法单元（tocken class)
    |        └── 手撸的正则表达式引擎（拓展的正则表达式->表达式树->NFA->DFA->最小化DFA)
    |             └── 最小化DFA
    |                 └── DFA
    |                     └── NFA
    |                         └── 表达式树 （递归下降分析）
    |                             └── 拓展的正则表达式（输入）
    └── 扫描器（scanner class）
